== Description ==

Este plugin permite habilitar o recurso do POPUP da sua plataforma SAC DIGITAL diretamente ao seu site. 

== Installation ==

Passos para instalação:

1. Faça o upload `sac-digital-popup` na pasta `/wp-content/plugins/`
2. Ative o plugin através do menu 'Plugins' no WordPress
3. Adicione as credenciais da autenticação do seu popup

== Frequently Asked Questions ==

= Onde encontro o token de acesso do popup? =

Acesse seu painel de atendimento, em Módulos > Integração > POP-UP, acesse a aba "Autenticação" e copie o **Token de Acesso** indicado na página.

= Habilitei e configurei, mas meu popup não aparece? =

Verifique as restrições que adicionou em seu painel de Gestor (domínio principalmente), caso ainda persista, contate suporte em https://suporte.alertrack.com.br
